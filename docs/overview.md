# 1. Tổng Quan Về LLM

## 1.1 LLM Là Gì?

Large Language Model là một loại mô hình deep learning được huấn luyện trên lượng dữ liệu văn bản khổng lồ để hiểu và sinh ra ngôn ngữ tự nhiên. LLM có khả năng:

- Hiểu ngữ cảnh và ý nghĩa của văn bản  
- Sinh ra văn bản mạch lạc và có ý nghĩa  
- Thực hiện nhiều tác vụ ngôn ngữ khác nhau mà không cần huấn luyện riêng  
- Học từ ít ví dụ (few-shot learning)

## 1.2 Ứng Dụng Thực Tế

- **Chatbot và trợ lý ảo**: ChatGPT, Claude, Gemini  
- **Dịch thuật**: Dịch tự động đa ngôn ngữ  
- **Tóm tắt văn bản**: Rút gọn tài liệu dài  
- **Sinh code**: GitHub Copilot, Code generation  
- **Phân tích cảm xúc**: Đánh giá sentiment trong text
